# Loïc Devaux — Artist Spotlight

Simple landing page for the Belgian painter Loïc Devaux.

## Tech stack

- Vite
- React + TypeScript
- Tailwind CSS

## Local development

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
```

Output goes to `dist/`.

## Deploy

Deployed via Vercel. Framework preset: Vite (auto-detected).
